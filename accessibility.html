Accessibility Statement

CPM Installer Services aims to keep this website welcoming, usable, and easier to navigate for as many visitors as possible.

Last updated: July 13, 2026

Our Approach

We aim to make the site practical and understandable for visitors using different devices and ways of navigating. Accessibility is an ongoing effort, and some content may still need improvement.

Practices We Use

Where practical, we build and review the site with these considerations in mind:

  • Alternative text for meaningful images.
  • Readable color contrast and clear visual hierarchy.
  • Keyboard navigation for interactive controls where it applies.
  • Semantic page structure and headings that help organize content.
  • Labels and instructions for the quote request form.

Need Help or Found a Barrier?

If you have difficulty using this site or need information in another format, please contact CPM Installer Services through the Contact page. Describe the page or task involved so we can better understand the issue.

Ongoing Improvement

We welcome feedback and may update this statement or the site as we make improvements. This statement describes our current approach and is not a promise that every feature will work perfectly in every situation.
